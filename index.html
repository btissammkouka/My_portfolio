Movie Reviews Sentiment Analysis

Project Information

Project Description

The Movie Review Sentiment Analysis project analyzes user reviews from the IMDb dataset to determine sentiment (positive or negative). Using Natural Language Processing (NLP) techniques, text data is preprocessed and transformed into feature representations. Four machine learning models—SVM, KNN, Naïve Bayes, and Random Forest—are trained and evaluated to compare their accuracy in classifying sentiments. The project aims to identify the most effective model for sentiment prediction, providing insights into audience opinions on movies.

Python NLTK TensorFlow Transformers NLP Streamlit
Movie Reviews Sentiment Analysis (From Scratch)

The Movie Review Sentiment Analysis project analyzes user reviews from the IMDb dataset to determine sentiment (positive or negative). Using Natural Language Processing (NLP) techniques, text data is preprocessed and transformed into feature representations. Four machine learning models—SVM, KNN, Naïve Bayes, and Random Forest—are trained and evaluated to compare their accuracy in classifying sentiments. The models were trained on this dataset, and their performance was assessed using metrics such as accuracy, precision, recall, and F1-score. The project aims to identify the most effective model for sentiment prediction, providing insights into audience opinions on movies.

Voice to Text : Real-Time Speech Recognition,Transcription, Translation, and Speaker Diarization

This project implements a real-time speech recognition system on a Raspberry Pi 3. It captures audio via USB or Bluetooth microphones, applies Meta's denoising algorithm for noise reduction, and processes it using Whisper for multilingual transcription. Speaker diarization is handled with PYANNOTE embeddings and K-Means clustering, while Google Translate enables real-time translation. The system ensures portability and IoT compatibility, making it suitable for various applications.
